### elliottprogrammer Level-Up game

Sort of like a game. Built with JavaScript and the HTML Canvas API.

Jump the elliottprogrammer cartoon character (an animated sprite) up the various platform levels.

Live demo at: [https://codepen.io/melliatto/pen/myVGorL](https://codepen.io/melliatto/pen/myVGorL);



Originally I built this for my [Dev Portfolio Website](https://elliottprogrammer.com).
